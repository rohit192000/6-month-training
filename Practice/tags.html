background-color:- It defined which color of the background you want to set.
color: change the color of the text.
height:- changes the height of the box.
width:- changes the width of an box.

Now lets study about CSS 
We can write css in three different forms:- 
1. inline css
2. internal css
3. external css

inline css 
In this we use style attribute to define css properties for given tag.
properties have to be written in double quotes""
Syntax:- <Tag start style="property name: property value;"> tag close
<div style="height: 100px; width:100px; background-color:red; color: black;">Hello</div>

Hello


You have been seeing a box around the text hello. This is your div container.
Now you can place all your section in a div and style them using inline css. Buty it is not recomended. 
External CSS(recomended)

Now let's go to the internal css 
For internal css we use style tag <style></style>

We write this tag inside our head tag.
How we write it lets start
<style>
    elemnent{
        property: value;
    }

    if you want the div to change the color of div to red using internal css then you have to write this code. 

    
        <head>
            <style>
                div{
                    background-color: red;
                }
            </style>
        </head>
    


    now lets go to external css.

    In this you create a external css file which have an extension .css. 
    it have a small difference than internal you don't have to use style tag in this 
    you can directly start with elemnt you want to provide css properties 
    then call this file in the html in head tag.
    contents in the file text.css  
    div{
        background-color: red;
    }

    h1{
        background-color: red;
    }


    Now call this file in html. Remeber both files have to be in the same folder. Eg. folder/html then also folder/text.css 
    <head>
        <link rel="stylesheet" href="text.css">
    </head>

    After this those css properties will aplly to the html file.
    Advantage:-  Now you can aplly same properties in whole html document.
    You don't have to write same property again and again.
    And anyone can use your css file if they want it. They just have call that file.
    So this method solve the disadvantages of both internal (cannot share with another document) and inline(increase length of the code by defining same property again and again) css

    Important :- inline>>internal>>external means same property if apllies in all method then the property
    value given in the inline css will apply other two will not apply. Same if same property given in internal and external css on same 
    element mwith different values then value of internal css will apply. Remember this.
